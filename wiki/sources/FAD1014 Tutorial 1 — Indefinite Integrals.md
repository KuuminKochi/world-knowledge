# Tutorial 1: Indefinite Integrals

Tutorial problems covering basic integration, substitution method, and integration by parts.

## Sections

### Basic Integration (Problems 1-4)
- Power rule applications
- Trigonometric integrals
- Exponential and logarithmic integrals
- Combining basic rules

### Integration by Substitution (Problems 5-8)
- Simple substitutions ($u = ax + b$)
- Trigonometric substitutions
- Exponential substitutions
- Recognizing derivative patterns

### Mixed Integration Techniques (Problems 9-12)
- Problems requiring multiple approaches
- Choosing appropriate methods

## Key Techniques Practiced

1. **Basic antiderivatives**: reversing differentiation rules
2. **Substitution**: identifying $u$ and $du$
3. **Pattern recognition**: spotting composite functions
4. **Algebraic manipulation**: rewriting integrands
