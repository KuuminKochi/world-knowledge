## Key Formulas

**Area under curve** (where $f(x) \geq 0$):
$$A = \int_a^b f(x)\,dx$$

**Area between curves** (where $f(x) \geq g(x)$):
$$A = \int_a^b [f(x) - g(x)]\,dx$$

**General area**:
$$A = \int_a^b |f(x) - g(x)|\,dx$$

## Steps for Area Between Curves

1. Find points of intersection (set $f(x) = g(x)$)
2. Determine which function is above the other
3. Set up the integral with proper limits
4. Evaluate
