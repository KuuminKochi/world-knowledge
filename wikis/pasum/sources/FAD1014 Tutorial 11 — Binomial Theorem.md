## Key Formulas

**Standard Binomial**:
$$(a + b)^n = \sum_{r=0}^{n} \binom{n}{r} a^{n-r}b^r$$

**General Term**:
$$T_{r+1} = \binom{n}{r} a^{n-r}b^r$$

**Binomial Coefficient**:
$$\binom{n}{r} = \frac{n!}{r!(n-r)!}$$

**General Binomial** (for any real $n$):
$$(1 + x)^n = 1 + nx + \frac{n(n-1)}{2!}x^2 + \cdots$$
