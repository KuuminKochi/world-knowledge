How to Use

For humans: Browse the wikis above. Each wiki has an index page with all concepts, sources, tutorials, and entity pages. Follow links between pages. Check the changelog to see what's new.

For AI assistants: Read the AI Navigation Guide first. When you land on a wiki index, check its recommended skills — these help you help the user better.

Each wiki follows a consistent structure: concepts (atomic knowledge), sources (lectures, chapters), tutorials (practice problems), and entities (people, courses, institutions). Wikilinks connect everything.
